Tentang Saya

Junior Web Developer

Saya adalah mahasiswa aktif semester 5 Program Studi Ilmu Komputer dengan fokus studi pada pemrograman, sekaligus seorang Junior Web Developer yang menaruh minat besar pada pengembangan Front End dan didukung dengan pemahaman Back End dasar. Saya terbiasa membangun antarmuka website yang modern, responsif, dan mudah digunakan dengan memperhatikan detail visual serta pengalaman pengguna. Selain pengembangan sisi tampilan, saya juga memiliki pemahaman dalam Back End seperti pengelolaan database, pembuatan API sederhana, serta penerapan logika aplikasi untuk mendukung fungsionalitas website. Dengan kombinasi kemampuan tersebut, saya mampu memahami alur kerja aplikasi secara menyeluruh, mulai dari perancangan antarmuka hingga proses pengolahan data. Saya memiliki semangat belajar yang tinggi, terbuka terhadap feedback, dan senang mengembangkan kemampuan teknis maupun non-teknis melalui praktik langsung. Melalui program magang, saya ingin terlibat dalam proyek nyata, belajar bekerja dalam tim pengembangan, serta memberikan kontribusi positif dalam pembuatan aplikasi web yang berkualitas.
